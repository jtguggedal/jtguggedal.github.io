Web Bluetooth / Characteristic Properties Sample

Available in Chrome 48+ | View Source | View on GitHub | Browse Samples

The Web Bluetooth API discovers and communicates with devices over the Bluetooth 4 wireless standard using the Generic Attribute Profile (GATT). It is currently only partially implemented in Chrome OS and Android M behind the experimental flag chrome://flags/#enable-web-bluetooth. Check out all Web Bluetooth Samples.

This sample illustrates the use of the Web Bluetooth API to display all properties of a specific characteristic from a nearby Bluetooth Low Energy Device. You may want to try this demo with the BLE Peripheral Simulator App from the Google Play Store.
